# Banner Images

This directory contains visual assets for Earshot.

## Required Images

- `banner-dark.png` - Dark theme banner (1200x400 recommended)
- `banner-light.png` - Light theme banner (1200x400 recommended)

## Creating Banners

Use any design tool (Figma, Canva, etc.) to create banners with:

1. "Earshot" title in a modern font
2. A waveform visualization graphic
3. Tagline: "Local Audio Transcription for macOS"

### Dark theme colors:
- Background: #1e1e1e
- Accent: #4a9eff
- Text: #e0e0e0

### Light theme colors:
- Background: #f5f5f5
- Accent: #0066cc
- Text: #1e1e1e
